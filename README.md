# 🚲 Bike Rental Management System

A console-based Bike Rental Management System developed using **Java**, **Object-Oriented Programming (OOP)**, and the **Collections Framework (ArrayList)**.

## ✨ Features

- Add Bike
- Remove Bike
- Display All Bikes
- Display Available Bikes
- Display Booked Bikes
- Search Bike
- Add Customer
- Display Customers
- Search Customer
- Book Bike
- Return Bike
- Display Active Bookings
- Dashboard
- Duplicate ID Validation
- Rental Days Validation
- Input Validation using Exception Handling

## 🛠️ Technologies Used

- Java
- OOP (Object-Oriented Programming)
- Collections Framework (ArrayList)
- Exception Handling
- Eclipse IDE

## 📂 Project Structure

```
src/
└── com/
    └── bikerental/
        ├── Bike.java
        ├── Customer.java
        ├── Booking.java
        ├── RentalService.java
        └── Main.java
```

## ▶️ How to Run

1. Clone this repository.
2. Open the project in Eclipse or IntelliJ IDEA.
3. Run `Main.java`.
4. Use the menu-driven interface.
